Inputs for the  SEAQT code (with example numbers)

1. Electron ev file (electron_ev.xlsx)
2. Electron dos file (electron_dos.xlsx)
3. Phonon ev file (phonon_ev.xlsx)
4. Phonon dos file (phonon_dos.xlsx)
5. Fermi Energy ((6eV) * 1.60218e-19 Joules)
6. Phonon group velocities (6000 m/s)
7. Phonon relaxation time (50e-12 seconds) 
8. Subsystems sizes (1e-7 meters for each subsystem)
# For the SEAQT phonon relaxation time, calculate using (((subsystem size)/(phonon velocity))^2 / (Phonon relaxation time))
# This is for all phonon ev amounts.

9. Number of subsystems (12)
10. Temperature of subsystems (295 Kelvin for the first 1-6, 300 Kelvin for the second 7-12)
11. Time (20*max(tau))
